#  Gerenciador de Cadastros

Este é um projeto de gerenciamento de nomes desenvolvido durante a minha transição de carreira para **Engenharia de Software**. O foco foi aplicar conceitos de lógica de programação, manipulação de arquivos e criação de interfaces gráficas modernas.

## 🚀 Tecnologias Utilizadas
* **Python 3.14**
* **CustomTkinter**: Para uma interface com design moderno e suporte a Dark Mode.
* **OS**: Para manipulação de caminhos e persistência de dados em arquivos locais.

## 🛠️ Funcionalidades
- **Cadastro Dinâmico**: Adiciona nomes à lista e valida duplicatas.
- **Persistência de Dados**: Todos os nomes são salvos automaticamente em um arquivo `nomes.txt`.
- **Interface Profissional**: Layout organizado com barra lateral (sidebar) e feedback visual de status.
- **Seleção Inteligente**: Ao clicar em um nome na lista, ele é enviado automaticamente para o campo de edição.

## 📂 Estrutura do Projeto
- `app.py`: Arquivo principal com a lógica e interface.
- `nomes.txt`: Banco de dados simples para armazenamento dos nomes.
- `.gitignore`: Configuração para manter o repositório limpo (ignorando pastas de ambiente virtual).
